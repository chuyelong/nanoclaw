# NanoClaw Capabilities

## Built-in

**Talk to Claude via WhatsApp**
- Ask questions, get answers, have conversations
- Full Claude capabilities: reasoning, writing, analysis, coding

**Web access**
- Search the web, fetch and read pages
- Research topics, summarize articles, check news

**File operations**
- Read, write, and edit files in mounted directories
- Great for managing notes, documents, code

**Scheduled tasks**
- "Remind me every Monday at 9am to review my tasks"
- "Every Friday, summarize the week and message me"
- "At 5pm today, send me a weather summary"

**Browser automation**
- Fill forms, take screenshots, interact with web apps
- Scrape data, automate repetitive web tasks

**Agent Swarms**
- Spin up teams of specialized sub-agents for complex tasks
- One agent researches, another writes, another reviews

**Multiple groups**
- Add other WhatsApp groups so different people/contexts get their own isolated agent with separate memory

## Skills

| Skill | What it adds |
|-------|-------------|
| `/add-telegram` | Telegram as a channel |
| `/add-gmail` | Read and send emails |
| `/add-voice-transcription` | Transcribe WhatsApp voice notes |
| `/x-integration` | Post tweets, like, reply on X |
